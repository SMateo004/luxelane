# Assets — Luxelane

Pon tus imágenes en las carpetas indicadas. El código las referenciará por nombre exacto.
Formato recomendado: **JPG** para fotos, **PNG** para logos/íconos con fondo transparente.

```
assets/
├── images/
│   ├── brand/          Logo, splash
│   ├── home/           Carrusel hero (pantalla principal web)
│   ├── vehicles/       Por cada clase de vehículo: carrusel (4 slides) + card thumbnail
│   │   ├── business/
│   │   ├── first_class/
│   │   ├── van/
│   │   └── electric/
│   ├── booking/
│   │   ├── luggage/    Opciones de equipaje (3 imágenes)
│   │   └── seating/    Opciones de asiento (5 imágenes)
│   └── auth/           Fondo panel izquierdo de login/registro
└── icons/
    ├── vehicles/       Íconos PNG de cada clase de vehículo
    └── ui/             Íconos custom de la interfaz
```

Ver el README.md dentro de cada subcarpeta para el nombre exacto de cada archivo.
